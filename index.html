为什么需要它

每天要在各种后台/表单里填大量字段——抖店、拼多多、1688、CRM、ERP、工单系统……大部分字段的含义并不直观，规则文档散落在十几个帮助页。

网页小助手把你的产品手册、帮助文档打包成本地知识库，在你填表时用 AI 告诉你"这个字段应该填什么"，并给出可一键填入的候选值。

核心特性

按需注入

默认不在任何网站工作，只有把站点加入白名单后才启用。

本地知识库

支持深度抓取、页面采集和本地文档导入（txt / md / html / docx / pptx / pdf）。

BM25 检索

中文友好，不需要 Embedding，零推理成本。

自带任意 LLM

默认 DeepSeek，也可填任何兼容 OpenAI Chat Completions 协议的服务。支持流式和思考模式。

AI 助手面板

侧边对话，支持全屏/区域截图作为上下文。对话按浏览器会话存储。

敏感字段过滤

密码、验证码、邮箱、手机号、证件号、银行卡自动跳过，不注入也不上传。

开始使用

  1. 从 Chrome 商店安装本扩展
  2. 进入"模型"设置页，粘贴你的 LLM API Key
  3. 进入"启用站点"，把要使用的域名加入白名单（Chrome 会弹权限确认）
  4. （可选）进入"知识库"，抓取站点帮助页或导入本地手册

隐私

本扩展没有任何自有服务器。所有配置、知识库、对话记录只保存在你本地浏览器里。只有当你主动点击"?"按钮或向 AI 助手发问时，扩展才会把字段上下文发到你自己配置的大模型服务，不经过我们。
