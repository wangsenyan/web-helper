网页小助手 — 使用文档

一、安装

  1. 打开 Chrome 浏览器，访问 Chrome Web Store 并安装本扩展（审核通过后提供链接）。
  2. 安装完成后，浏览器右上角会出现扩展图标，建议点"固定"。
  3. 安装后会自动打开设置页。

二、配置模型（必做）

  1. 在设置页点"模型"tab。
  2. 默认卡片是 DeepSeek。在 platform.deepseek.com/api_keys 申请 API Key 后粘贴进来。
  3. 建议选择 deepseek-v4-flash 模型，思考模式 禁用（响应更快）。
  4. 勾选"流式返回"可以边生成边显示。
  5. 点"测试连接"，看到绿色"连接成功"即 OK。
  6. 点"保存"。
提示： 你也可以添加多个模型配置，比如同时配置 DeepSeek 和 OpenAI 作为备份。Base URL 留空默认是 https://api.deepseek.com；如果用其他兼容 OpenAI 协议的服务（Moonshot、智谱、百炼、Groq、Ollama、OpenAI），改 Base URL 和模型名即可。

三、启用站点（必做）

  1. 点"启用站点"tab。
  2. 在白名单文本框里每行输入一个域名，例如： 
    work.1688.com
seller.douyin.com
  3. 点"保存"。Chrome 会弹权限确认框，一定要点"允许"。
  4. 刷新目标页面，即可看到"?"按钮。
为什么需要授权？ 本扩展默认不访问任何网站。只有你主动把站点加入白名单、Chrome 弹窗确认后，扩展才能在该站点注入脚本。这是为了保护你的隐私。

四、知识库（可选但强烈推荐）

知识库让 AI 回答更准确。三种建库方式：

方式 1：深度抓取

  1. 点"知识库"tab → "新建知识库（深度抓取）"。
  2. 填知识库名称（比如"抖店帮助文档"）。
  3. 填起始 URL（建议是站点的帮助中心首页）。
  4. 最大深度建议 2，最大页数建议 50。
  5. 公开页面选"公开抓取"；需要登录后才能看的后台页选"登录态抓取"（会带你当前浏览器 Cookie）。
  6. 点"开始抓取"，等进度条跑完。

方式 2：采集当前渲染页面

对抖店学堂、飞书文档这类动态渲染的 SPA 效果更好：

  1. 在浏览器里打开目标页面，等内容加载完。
  2. 回到设置页，点"采集当前已渲染页面"。

方式 3：导入本地文档

  1. "知识库"tab → "导入本地文档"卡片。
  2. 填知识库名称和关联站点 URL。
  3. 选择文件（支持 txt / md / html / json / csv / docx / pptx / pdf），可多选。
  4. 单文件 ≤ 10 MB，总大小 ≤ 20 MB。
  5. 点"导入文档"。
文档内容 100% 在浏览器本地解析，不会发送到任何服务器。敏感信息会自动脱敏。

五、日常使用

字段填写建议（"?"按钮）

在任何授权站点的表单输入框旁会看到橙色"?"按钮。点击它会弹出：

  • 含义：这个字段是干什么的
  • 应填内容：格式、范围、注意事项
  • 候选值：1-3 个可直接点击填入的值
  • 参考来源：知识库中的相关页面链接

AI 助手面板

页面右下角有一条竖边"AI 助手"，点击展开：

  • 直接在输入框里问任何和当前页面相关的问题
  • "全屏截图"：把整个页面作为上下文
  • "区域截图"：框选页面的局部作为上下文
  • "清空"：清空当前会话

截图作为附件挂在输入框旁，发送后连同问题一起发给 AI。

右键菜单

在页面上选中一段文字，右键 → "AI 搜索：<你选的文字>"，可以带上下文直接发问。

六、隐私保护

  • 密码、验证码、邮箱、手机号、证件号、银行卡、Token 等字段自动跳过，不注入"?"按钮也不发送给模型。
  • 发送给模型的文本会先脱敏：邮箱/手机号/身份证/银行卡/API Key 会被替换成占位符。
  • 所有配置、知识库、对话记录只保存在本地浏览器。
  • 关闭浏览器后会话级对话自动清除。

七、常见问题

"?"按钮没出现？

  1. 确认域名在白名单里。
  2. 确认已刷新页面（给扩展一次重新注入的机会）。
  3. "数据"tab → 点"诊断当前活动标签页"，看输出：
    • enabled: false → 去启用站点 tab 添加
    • totalInputs: 0 → 页面真没输入框，或都在 iframe，刷新试试
    • injected: 0 → 按钮被页面元素遮挡或过滤了
  4. 密码、邮箱、手机号等敏感字段不会有按钮，这是设计如此。

点"?"报"未配置 API Key"？

去"模型"tab 填 Key，点保存，再试。

点"?"很久没反应？

网络原因或模型服务有延迟。勾选"流式返回"可以看到生成过程。

对话历史丢失？

关闭浏览器时会话级对话会被清除（这是隐私设计）。如果你只是刷新页面，对话应该还在。

知识库抓取失败？

常见原因：

  • 目标站点是 SPA（单页应用），初始 HTML 为空 → 改用"采集当前已渲染页面"
  • 需要登录 → 把抓取模式改成"登录态抓取"
  • 被站点 robots.txt 或反爬机制拒绝 → 换起始 URL 或改用本地文档导入

卸载后数据怎么处理？

Chrome 卸载扩展时会自动清空 chrome.storage.local 和 chrome.storage.session，所有本地数据会被删除。
