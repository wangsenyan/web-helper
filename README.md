# 网页小助手 · 官网与文档

这是 [网页小助手 Chrome 扩展](https://wangsenyan.github.io/web-helper/) 的官网与文档站点。本仓库**只包含公开网站内容**（隐私政策、使用文档），不含扩展源代码。

## 内容

- `index.html` — 官网首页
- `privacy.html` — 隐私政策（Chrome Web Store 审核用）
- `usage.html` — 使用文档

## 通过 GitHub Pages 发布

1. 把此目录下所有文件（不含 README.md 也行）推到 `main` 分支根目录
2. Repo Settings → Pages → Source: Deploy from branch, Branch: `main` / `/ (root)`
3. 访问 `https://你的用户名.github.io/仓库名/`
